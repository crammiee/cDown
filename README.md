# cdown
**A tiny command-line helper script that simplifies downloading MP3 or MP4 files using yt-dlp.**

Instead of typing long commands, you can run:
```
cdown mp3
```
or
```
cdown mp4
```
Then paste your link... that’s it.

# Features
- Short, clean commands (cdown mp3, cdown mp4)
- Asks for the video URL interactively
- Passes correct format arguments to yt-dlp
- Works in any folder (downloads to current directory)
- Lightweight and portable

# Installation
1. Install yt-dlp (if you haven’t already)

&nbsp;&nbsp;&nbsp;&nbsp;https://github.com/yt-dlp/yt-dlp

&nbsp;&nbsp;&nbsp;&nbsp;Make sure ```yt-dlp``` is in your system PATH.

2. Clone this repo
Click Code → Download ZIP, or:
```
git clone https://github.com/YOURUSERNAME/cdown.git
```

3. Add cdown to your PATH
   Add the folder to your PATH:
   1. Search Environment Variables
   2. Edit Path
   3. Add the folder containing cdown.bat

# Usage
Navigate to the folder where you want the file downloaded, then open CMD (type cmd in File Explorer address bar).

***Download MP3***
```
cdown mp3
```
***Download MP4***
```
cdown mp4
```
Then paste your YouTube link when prompted.
